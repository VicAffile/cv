# Projet : Portfolio
Type : Site Web  
Langages : HTML 5, CSS 3 et JavaScript

# Description du projet  
L'objectif était de me créer un portfolio pour facilité ma recherche d'alternance et de stage. Je souhaitais quelque chose de personnel et de travailler pour que les personnes le consultant voient qui je suis.  
J'ai choisi de n'utiliser aucun framework pour monter mon savoir faire.
